# AWS AI Practitioner Exam Study Guide

This repository contains structured study notes for the AWS Certified AI Practitioner (AIF-C01) exam, built using MkDocs Material.

## How to use this guide

The guide is organized into several key areas:
- **ML Fundamentals**: Core concepts, supervised/unsupervised learning.
- **Deep Learning**: Neural networks and specialized architectures.
- **Generative AI**: Foundation models, RAG, and Amazon Bedrock.

## Local Development

If you want to run this documentation locally:

1. **Clone the repository:**
   ```bash
   git clone <repo-url>
   cd <repo-name>
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the development server:**
   ```bash
   mkdocs serve
   ```
   The site will be available at `http://127.0.0.1:8000`.
